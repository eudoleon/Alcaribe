POS-Point of sales Manager Validation Odoo Apps

This Odoo apps is useful for validate and authenticate pos manager for add/remove quantity, change price, apply discount, remove order line, remove order, payment and close pos screen, also manager authenticate one time password for pos order. which gives point of sale pos manager to override functions of pos with his approval to authorize in pos Odoo.

Features

Validation/Authentication on Quantity

Apply validation and authentication on add/remove quantity.

Validation/Authentication on Order/Orderline

Apply validation and authentication on remove pos order and orderline.

Validation/Authentication on Closing POS

Apply validation and authentication on close pos screen.

Warning Arise on Wrong Password

If user entered password is wrong will be arise warning.

User/Manager Configuration Pos Validation.

User can configure pos validation and authentication from back-end.

Validation/Authentication on Price.

Apply validation and authentication on change product price.

Validation/Authentication on Discount.

Apply validation and authentication on product discount.

Validation/Authentication Password/One Time Password on Order.

Apply validation and authentication for every time one time password on order.
