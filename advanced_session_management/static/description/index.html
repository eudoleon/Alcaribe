User Activity Audit (Advanced)

"Track Every User's Activities, Operations & Sessions of the System"

This Module Allows privileged users to track all the operations (such as read, create, modify and delete) performed on Data Models by the users & all their running sessions. And trace-back the session on which any perticular activity is performed. And also if any suspicious session started, the user can track down the activities (such as record read, create, update & delete) with a complete audit trail. And kill that session to force logout the malicious user. This module will kill user's sessions after a specified time duration if defined in configuration settings. This module will automatically notify the respective users by mail for any new login.

Why should I use this app?

This module will add extra layer of security on your ERP. It will send you notification of any unwanted sign-ins, it will help you to trace the activities performed during that session and most important you can kill that unwanted session with just one click.

What is User Audit/Activity Log?

User audit/activity log is collection of data from the actions performed by user in any session, such as- create a record, update any field, update any view, delete any record, etc.

How can I clear the log data?

The module is set to clear the log data automatically after 7 days. You can configure it from settings.
