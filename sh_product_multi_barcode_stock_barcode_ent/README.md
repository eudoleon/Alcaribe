About
============
Mainly the use of this module, you can assign multiple Barcode/QRCode for each product. Also, you can search that product in inventory operations like the delivery order, incoming order (receipt ), inventory adjustment, internal transfer & return order.

Installation
============
1) Copy module files to addon folder.
2) Restart odoo service (sudo service odoo-server restart).
3) Go to your odoo instance and open apps (make sure to activate debug mode).
4) click on update app list.
5) search module name and hit install button.
