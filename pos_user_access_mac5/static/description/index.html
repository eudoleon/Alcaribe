[Original] POS User Access

Way to manage access for specific features in POS for each user

Features:

  • User Access to POS Closing
  • User Access to Order Deletion
  • User Access to Order Line Deletion
  • User Access to Order Payment
  • User Access to Discount Application
  • User Access to Price Change
  • User Access to Decreasing Quantity

Configuration

Easy setup of user access to POS

User access configuration is as easy as ABC. Configuration can be found in Settings > Users & Companies > Users.

For each POS users, specify which POS feature should be accessible or restricted in POS User Access tab.

POS Interface

User access/restriction seemless integration to POS interface

There will be an error prompt to the user if the user has no access to a certain feature.

Compatibility

This module is compatible with Odoo 16.0 Community and Enterprise Editions
